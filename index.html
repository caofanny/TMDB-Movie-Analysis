Introduction to project, including the task and topics (5pts)

Dataset Introduction
The dataset we used is called the TMDB 5000 Movie Dataset, which contains metadata on 4800 movies from Kaggle. This dataset contains information from The Movie Database (TMDb) API. It includes 20 features for data like budget, revenues, genres, popularity, and production company, which covers both categorical (genre) and continuous (budget) data. We will analyze how various features of movie production may contribute to a movie's overall success and returns. Since it is in a CSV file, we will be loading it using Pandas for cleaning and manipulation as part of data visualization.
